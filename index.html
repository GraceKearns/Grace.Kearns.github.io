Projects

    Word Guess

    "Word Guess" is a captivating mobile application inspired by the classic game WORDLE. However, it offers a unique twist: instead of guessing a five-letter word, players are presented with a word's definition and tasked with deducing the corresponding word. This engaging word game challenges users to apply their vocabulary and problem-solving skills in an enjoyable and intellectually stimulating manner.

    Google Drive Backup

    "Google Drive Backup" A python desktop application that connects to a Google User's Google drive and allows them make a backup locally on the user's desktop. This application broadens itself to download multiple files at a time and can download entire folders in two clicks.

    Cloud Decision Tool

    The CDT (Cloud Decision Tool) Web tool is a front-end web application written on the React Javascript Framework on top of the create-react-app environment. This tool was developed to allow users to gain an understanding of their unknown cloud system requirements through a questionnaire process. Unknown requirements of the system are inferred through our clients Case-Based Reasoning (CBR) algorithm. The system is built with mostly the React and Material-UI frameworks that interact with the previously developed CDT-CBR micro-service (for implementation of CBR technique) and a database (for storing of various case data).
